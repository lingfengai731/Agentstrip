## 🤔 这是什么？

你有没有过这样的经历？
- 想去巴厘岛玩 7 天，但不知道从哪开始计划
- 在小红书翻 3 小时还没找到符合预算的酒店
- 问 ChatGPT 它给你一堆笼统建议，没有真实价格、没有当地时区、没有实时天气
- 看完攻略，转头就忘了哪个景点适合自己

**WanderMind 解决的就是这个问题。**

它不是一个聊天机器人，而是一个**有 6 位专业 AI 智能体的旅行团队**：
- 🗺️ **旅程规划师** 设计每日行程和路线
- 🏨 **住宿顾问** 推荐酒店并查实时价格
- 🍜 **美食探索家** 找当地隐藏餐厅
- 🏄 **活动策划师** 推荐独特体验
- 💰 **预算管家** 精确估算费用
- 🔍 **实时搜索员** 抓最新签证 / 天气 / 价格

它们会**记住你的旅行偏好**（你喜欢豪华还是经济、独自出行还是带孩子、喜欢文化还是冒险），让每次对话都越来越懂你。

## ✨ 你能用它做什么

### 1. 📋 让 AI 团队为你做完整规划
告诉它"我想去巴厘岛 7 天，2 人 1 万 5 预算"，几秒钟内得到：
- 每日详细行程（含景点、餐厅、住宿）
- 真实酒店价格（直接跳转预订）
- 预算分配建议
- 当地实用贴士（签证、礼仪、安全）

### 2. ⚡ 一键问全队
对同一个问题，让 **3 位专家同时作答**（真正的并行处理，3 秒内出结果），分别从规划、活动、预算视角给你建议。

### 2️⃣b ⚡/🎯 快慢双轨模式
输入框旁边的小开关：
- **⚡ 极速**（默认）—— Qwen2.5-7B 闪电响应，**首字 200ms** 内出，适合日常问答和快速试探
- **🎯 精细** —— MiMo Pro 深度思考，回答更细腻完整，适合做正式行程规划

每条 AI 消息上会有徽章告诉你这条是用哪个模型生成的。**选择记忆在浏览器，下次自动恢复。**

### 3. 🌍 输入任何城市，自动加载当地信息
不止巴厘岛 / 京都 / 巴黎 / 圣托里尼——输入"冰岛雷克雅未克"或"摩洛哥马拉喀什"，AI 自动生成：
- 当地实时天气和**精确时区时钟**
- 推荐区域（4 个）和旅行贴士（3 条）
- 当前汇率参考

### 4. 💰 一站式比价 Tab（酒店 + 机票）
右栏专门的 **💰 比价 Tab**，里面有酒店和机票两个子页签，一处搞定行程两大刚需开销。

#### 🏨 酒店比价 + 区域筛选
- **可选住宿区域**：每个目的地都有热门住宿区 chips 横排筛选
  - 巴厘岛 9 区：水明漾 / 仓古 / 乌布 / 乌鲁瓦图 / 沙努尔 / 金巴兰 / 库塔 / 努沙杜瓦 / 蓝梦岛
  - 京都 5 区：祇园 / 河原町 / 东山 / 嵐山 / 京都站
  - 巴黎 6 区：玛黑区 / 圣日耳曼 / 蒙马特 / 拉丁区 / 香榭丽舍 / 卢浮宫
  - 圣托里尼 5 区：伊亚 / 菲拉 / 伊姆罗维利 / 卡马里 / 佩里萨
  - **任意自定义目的地**：AI 自动生成 6 个真实热门区域
- 选好区域 + 入住/退房日期 + 人数，搜索 Google Hotels 真实房价
- 卡片显示评分、缩略图、设施、点击直达预订页

#### ✈️ 机票比价
- **出发城市**：默认上海 PVG，下拉支持北上广深成港杭西 8 大城市 + ✨ 其他城市自由输入
- 往返/单程 toggle、起飞日期、返回日期、人数
- 搜索 Google Flights：航空公司 logo、真实票价、转机次数（直飞绿色 / 1 转中黄色 / 多转红色）、总时长、直达预订链接

### 5. 🌌 平行宇宙预览
同一段旅行，AI 同时生成 **3 套截然不同的方案**（节俭穷游 / 舒适平衡 / 奢华享受），选你喜欢的那个开始规划。

### 6. 🗺️ 探索地图
可视化 SVG 地图，景点 / 美食 / 住宿 / 活动按颜色分类，**每个地点带 24 小时人流预测热力图**，避开拥挤时段。

### 7. 🧮 智能预算计算器
按目的地、天数、人数、风格、出发城市，估算机票 / 住宿 / 餐饮 / 活动 / 交通的全程预算。

### 8. ✏️ AI 行程修改助手
行程生成后，AI 回复下方会自动出现 4 个修改建议：
- ✏️ **修改某一天** — 改第 3 天的安排
- ⏰ **调整节奏** — 太累了，能放松点吗
- 💰 **调整预算** — 降一档或升一档
- 🔄 **换景点 / 餐厅** — 不喜欢这个，换个类似的

AI 会**只改你提到的部分**，保留其他不变，并用 ✏️ 标出改动处。再也不用每次都从头重新规划。

### 9. 📝 一键生成旅行游记
和 AI 聊完一次完整规划后，点击"游记生成"，它会基于你们的对话生成一篇第一人称的旅行游记（带小标题、感悟），方便分享或留念。

### 10. 📄 导出 PDF 行程
精美排版的行程单 PDF，可打印或转发给同行的朋友。

### 11. 🧠 旅行偏好记忆
保存一次偏好（预算档次、风格、同行方式），之后每次 AI 对话都会自动参考，不用反复说明。

## 📱 怎么使用？

### 电脑用户
1. 打开 **[https://agentstrip.onrender.com](https://agentstrip.onrender.com)**
2. 邮箱注册（10 秒）
3. 选择目的地或输入任意城市
4. 在对话框输入你的需求，AI 会一边搜索一边回答

### 手机用户
1. 用浏览器（Safari / Chrome / Edge 都可以）打开 **[https://agentstrip.onrender.com](https://agentstrip.onrender.com)**
2. iPhone：点击底部分享按钮 → 添加到主屏幕
3. Android：浏览器菜单 → 添加到主屏幕
4. 之后从主屏幕图标启动，体验和 App 一样

### 想尝试一下但不知道怎么开始？
注册后试试这些问题：
- "我想去巴厘岛蜜月，7 天，2 人预算 2 万，最浪漫的方案"
- "京都 5 天，喜欢传统文化和美食，避开人多的地方"
- "推荐巴黎的精品民宿，预算中等"
- 点击界面上的 **⚡ 问全队** 按钮，让 3 位专家同时给你建议
