## 🤔 这是什么？

你有没有过这样的经历？
- 想去巴厘岛玩 7 天，但不知道从哪开始计划
- 在小红书翻 3 小时还没找到符合预算的酒店
- 问 ChatGPT 它给你一堆笼统建议，没有真实价格、没有当地时区、没有实时天气
- 看完攻略，转头就忘了哪个景点适合自己

**WanderMind 解决的就是这个问题。**

它不是一个聊天机器人，而是一个**有 6 位专业 AI 智能体的旅行团队**：
- 🗺️ **旅程规划师** 设计每日行程和路线
- 🏨 **住宿顾问** 推荐酒店并查实时价格
- 🍜 **美食探索家** 找当地隐藏餐厅
- 🏄 **活动策划师** 推荐独特体验
- 💰 **预算管家** 精确估算费用
- 🔍 **实时搜索员** 抓最新签证 / 天气 / 价格

它们会**记住你的旅行偏好**（你喜欢豪华还是经济、独自出行还是带孩子、喜欢文化还是冒险），让每次对话都越来越懂你。

## ✨ 你能用它做什么

### 1. 📋 让 AI 团队为你做完整规划
告诉它"我想去巴厘岛 7 天，2 人 1 万 5 预算"，几秒钟内得到：
- 每日详细行程（含景点、餐厅、住宿）
- 真实酒店价格（直接跳转预订）
- 预算分配建议
- 当地实用贴士（签证、礼仪、安全）

### 2. ⚡ 一键问全队
对同一个问题，让 **3 位专家同时作答**（真正的并行处理，3 秒内出结果），分别从规划、活动、预算视角给你建议。

### 3. 🌍 输入任何城市，自动加载当地信息
不止巴厘岛 / 京都 / 巴黎 / 圣托里尼——输入"冰岛雷克雅未克"或"摩洛哥马拉喀什"，AI 自动生成：
- 当地实时天气和**精确时区时钟**
- 推荐区域（4 个）和旅行贴士（3 条）
- 当前汇率参考

### 4. 🏨 查真实酒店价格
直接在面板里选入住日期、退房日期、人数，一键搜索 Google Hotels 的真实房价，看评分、看图片、点击直达预订页。

### 5. 🌌 平行宇宙预览
同一段旅行，AI 同时生成 **3 套截然不同的方案**（节俭穷游 / 舒适平衡 / 奢华享受），选你喜欢的那个开始规划。

### 6. 🗺️ 探索地图
可视化 SVG 地图，景点 / 美食 / 住宿 / 活动按颜色分类，**每个地点带 24 小时人流预测热力图**，避开拥挤时段。

### 7. 🧮 智能预算计算器
按目的地、天数、人数、风格、出发城市，估算机票 / 住宿 / 餐饮 / 活动 / 交通的全程预算。

### 8. 📝 一键生成旅行游记
和 AI 聊完一次完整规划后，点击"游记生成"，它会基于你们的对话生成一篇第一人称的旅行游记（带小标题、感悟），方便分享或留念。

### 9. 📄 导出 PDF 行程
精美排版的行程单 PDF，可打印或转发给同行的朋友。

### 10. 🧠 旅行偏好记忆
保存一次偏好（预算档次、风格、同行方式），之后每次 AI 对话都会自动参考，不用反复说明。

## 📱 怎么使用？

### 电脑用户
1. 打开 **[https://agentstrip.onrender.com](https://agentstrip.onrender.com)**
2. 邮箱注册（10 秒）
3. 选择目的地或输入任意城市
4. 在对话框输入你的需求，AI 会一边搜索一边回答

### 手机用户
1. 用浏览器（Safari / Chrome / Edge 都可以）打开 **[https://agentstrip.onrender.com](https://agentstrip.onrender.com)**
2. iPhone：点击底部分享按钮 → 添加到主屏幕
3. Android：浏览器菜单 → 添加到主屏幕
4. 之后从主屏幕图标启动，体验和 App 一样

### 想尝试一下但不知道怎么开始？
注册后试试这些问题：
- "我想去巴厘岛蜜月，7 天，2 人预算 2 万，最浪漫的方案"
- "京都 5 天，喜欢传统文化和美食，避开人多的地方"
- "推荐巴黎的精品民宿，预算中等"
- 点击界面上的 **⚡ 问全队** 按钮，让 3 位专家同时给你建议
